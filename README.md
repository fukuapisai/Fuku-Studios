# FukuXyz - AI Enterprise Website

Website perusahaan teknologi AI dengan tampilan modern, futuristik, dan profesional setara startup AI kelas dunia.

## Fitur Utama
- Desain light mode premium dengan aksen biru neon & ungu
- Layout full responsive (mobile, tablet, desktop)
- Animasi CSS & JavaScript yang smooth
- Glassmorphism effect pada cards
- Form contact dengan backend API
- Navigation dengan smooth scroll
- Performance optimized

## Teknologi yang Digunakan
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js + Express
- Deployment: Vercel (serverless)
